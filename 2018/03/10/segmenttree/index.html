基本操作

  1. Segment Tree 是一个 Full Binary Tree，每个节点子节点数量为 0 或 2.
  2. 给定含 n 个元素的数组区间，对应的 Segmeng Tree 节点数量最多为 2n - 1.
  3. 偶数长度区间会拆出两个偶数OR奇数长度区间；奇数长度区间会拆出一个偶数+一个奇数长度区间， 最终以长度为 1 的区间为叶节点。
  • Build O(n)
  • Update O(log n)
  • Query O(log n)
