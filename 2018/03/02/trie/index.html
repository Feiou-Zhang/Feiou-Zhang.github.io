Template

HashMap trie:

  • 优点

    • 支持所有 Character
    • 相对更省空间

  • 缺点

    • 查询时间相对长
    • 尤其是有 wildcard 做 DFS 时

Array trie:

  • 优点
    • 查询速度快，尤其是有 wildcard 做 DFS 时
  • 缺点
    • 每个 node 空间占用大
    • 比较依赖指定字符集，比如 ‘a-z’ 这种
